# GuideTI-Variacao-do-Ativo

Solução desenvolvida utilizando .NET 6.0. Sendo assim se faz necessário o download do ".NET Runtime" ou ".NET SDK", conforme o ambiente em que a solução estiver operando.
O download pode ser realizado através do link: https://dotnet.microsoft.com/en-us/download

Para armazenar os dados, utiliza-se o Banco de Dados SQLite para facilitar o acesso aos dados sem a obrigatoriedade de instalação de outros softwares.
Caso o usuário deseje acessar os dados diretamente pelo arquivo com extensão '.sqlite', pode-se fazer um download de algum SGBD como o "DB Browser for SQLite". 
Link para o download da ferramenta: https://sqlitebrowser.org/dl/
